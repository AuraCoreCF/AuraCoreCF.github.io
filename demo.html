Demonstration Layer

A live evaluation path built for both technical and non-technical audiences.

AuraCoreCF is meant to be shown, not merely described. This page is already structured around the exact demo split the system calls for: one path for researchers and engineers who want field-state visibility and one path for outside audiences who need to feel the continuity, personality, and difference directly.

Technical track

Show the architecture operating in real time.

Open cold. Report the current field state. Show the JSON output live. Explain the seven fields and display the coherence number at baseline. Then push the system through a substantive technical question, an identity-protection challenge, and a new-session continuity check so the audience can see the structure move rather than merely hear claims about it.

Non-technical track

Show the experience before the theory.

Open warm. Let Aura greet by name. Ask what is remembered. Invite a self-generated story. Ask what makes it different from a conventional assistant. Close by showing the coherence number and giving one sentence on what that number means. The goal is clarity, not jargon.
